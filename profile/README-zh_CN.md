### 什么是 Module Federation?

Webpack Module Federation (MF) 是 webpack 的一个特性，允许从多个独立的构建系统动态加载一个模块的多个版本。

这样就允许创建微前端风格的应用程序，其中多个系统可以共享代码并动态更新，而无需重新构建整个应用程序。

它还允许具有不同发布周期的分布式团队和应用程序共享代码，而无需等待所有系统都同意并部署某个模块的单个共享版本。

此外，它允许基于路由和其他标准进行代码分割，这可以提高性能。

### MF 书籍

[Practical Module Federation](https://module-federation.myshopify.com/products/practical-module-federation) - Webpack Module Federation官方书籍由 Jack Herrington & Zack Jackson所编写。《Practical Module Federation》是第一本也是唯一一本关于Webpack 5创新的实时代码共享机制的书籍。它带你完成从Module Federation开始所需要做的所有事情。它涵盖了Module Federation的内部实现，以及模块联邦如何与其他共享选项相匹配。这本书还涵盖了许多实际的主题，包括：跨共享代码的状态共享、不同的部署选项、共享非视图相关的代码、编写代码以应对代码和网络故障，等等。
